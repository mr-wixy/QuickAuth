| Typecho、WordPress插件普通版 | 支持     | 支持     |
| 自定义小程序码图标           | 不支持   | 支持     |
| Typecho、WordPress插件高级版 | 不支持   | 支持     |
| 微信扫码登陆(H5)             | 不支持   | 支持     |
| 微信客户端静默登陆           | 不支持   | 支持     |
