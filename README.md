# QuickAuth

## 介绍

QuickAuth是一个免费、便捷的第三发登录平台

该仓库为QuickAuth的文档站

## 支持的第三方登录平台

- 微信（H5、开放平台、小程序）
- 短信验证码
- QQ
- Github
- Gitee
- 支付宝
- 微博
- 钉钉

## 软件特点

1. 简单易用，接口文档清晰完善，并提供Typecho WordPress插件
2. 接入免费，个人可免费接入，不需要企业资质，不需要进行微信认证
3. 支持定制，微信登录可自定义小程序图标，可自定义扫码登录页面等
